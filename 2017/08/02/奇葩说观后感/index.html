奇葩说观后感

Aug 2, 2017 | 杂

　　这段时间恶补了一下奇葩说第四季,不得不说确实很精彩,往往看似很简单的命题,在选手及导师们的解读下,常常会有不同角度,不同观点的精彩辩论,但是,经过这么多季的运作之后,也渐渐的涌现出了一些套路,让作为观众的我有些不舒服,也少了一些新鲜感,当然我自己也觉的有点过于强求,不论是谁都知道不可能有哪些节目会永远有新意的,有些应该不能说是套路,只是战术的一种,比如取出命题中的关键字,然后对其进行名词解释,再放到对应的情境中进行解读,这应该是很通用的一种辩论方法.我只是想把觉得最不舒服的套路说出来,那就是选手们经常会说的”对方辩友都在射程之内”
　　我本身不反感这种战术,或者说辩论方法,但是你见过打战的时候,当敌军冲过来时,有士兵从战壕里跳出来说”你在我的射程之内,我要射了”的吗?你开枪就是了,废什么话啊,这句话明显就是画蛇添足,而且如果我是对方辩友,我会感觉不舒服,稍稍破坏了大家一片友好的假象,同样也会给自己带来一些负面影响,至少我就觉得不爽.
　　另外我觉的这种方法被滥用了,变的一点技术含量都没有,因为他们常常用的一种射击方式,连小学生都会的,就是找同义词或近义词,举个例子如果对方说了一些自己遇到的困难,那么他就会说”你说的那些是困难吗?不是,那叫磨~~~~炼”,于是就成了他射程范围内了.再比如对方说了一些关于自己很寂寞的话,那么他就会说你那不是寂寞,那叫孤独或者孤单.照这个逻辑全世界都在射程之内吧.那你好好辩就是了,为啥还要先声报一下全世界都知道的事?你试想一下,如果有个人跟着你,然后每次吃饭之前总要跟你说一句”我们现在做的事叫吃饭”,做车之前跟你说一句”我们现在要坐的东西叫汽车”,睡觉的时候跟你说一句”我们接下来做的事叫睡觉”,唉..好烦!!!
　　总之节目很不错,但是希望之后不要再老说射程之内了,每次听道这句话都有想打人的冲动.
